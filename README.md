# Big Bird

This is the companion repository for Big Bird, which provides practical global privacy guarantees for the [W3C Attribution API](https://w3c.github.io/attribution/).
The paper is available on [arXiv](https://arxiv.org/abs/2506.05290).

Experimental code will be released soon. Prototypes are already available:
- Rust library for on-device DP budgeting: [columbia/pdslib](https://github.com/columbia/pdslib)
- Firefox integration of pdslib: [columbia/pdslib-firefox](https://github.com/columbia/pdslib-firefox)
